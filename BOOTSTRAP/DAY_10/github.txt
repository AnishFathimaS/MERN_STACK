==> Cloning Files from github to local

==> Commands

    ~ git clone "link"
    ~ cd demo 
    ~ git add filename (git add index.txt)
    ~ git commit -m 'commited'
    ~ git push origin main
    ~ git remote set-url origin https://<your-username>@github.com/AnishFathimaS/demo.git(put this cmd if suppose get error on git push)
    ~ git credential reject https://github.com (put this cmd if suppose get error on git push)
    ~ git pull 

===> Cloning files from local to github

===> Commands

    ~ git init 
    ~ git remote add origin https://github.com/YourUsername/YourRepository.git
    ~ git add 'filename' or git add .
    ~ git commit -m 'committed'
    ~ git push origin main (git push -u origin main)
